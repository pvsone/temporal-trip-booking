# Temporal Trip Booking - Java

## Prerequisites

- **Java 21** or higher
- **Gradle** (wrapper included)

## Build

Build the project:

```bash
./gradlew build
```

## Run

```bash
./start_worker.sh
```

Or directly with Gradle:
```bash
./gradlew run
```

## Test

Run the test suite:

```bash
./gradlew test
```
